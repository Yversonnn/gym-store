# Gym Store

A modern e-commerce web application for gym equipment and fitness products.

## Repository Information

- **Repository URL**: https://github.com/Yversonnn/gym-store
- **Branch**: main
- **Status**: Public

## Project Structure

- `frontend/` - React-based frontend application
  - `src/components/` - Reusable UI components
  - `src/pages/` - Application pages
  - `src/store/` - State management
  - `src/data/` - Product data

## Technologies Used

- React.js
- JavaScript
- CSS

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Yversonnn/gym-store.git
   ```

2. Navigate to the frontend directory:
   ```bash
   cd gym-store/frontend
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```
